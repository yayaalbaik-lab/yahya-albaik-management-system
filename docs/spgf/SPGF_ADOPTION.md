https://github.com/yayaalbaik-lab/spgf-general-standards

Until a corrective constitution release is produced, `v6.2` should be treated as the adopted generalized project-neutral version, but not as a complete replacement for all governance content that existed in `v6.1`.

Project decisions must avoid relying on unstable numeric section references from `v6.2`.

## Project Profile

Initial profile: `Standard`

## Rationale

The project is expected to include inventory, prices, purchasing, operational reports, and user permissions. These characteristics require more governance than `Lean`, but do not currently require full `Enterprise` controls.

## Rule

Files under `docs/spgf/general/` are reference copies and should not be edited for project-specific decisions. Project-specific application belongs under `docs/spgf/project/`.

# Out Of Scope

This document protects the first phase from uncontrolled expansion.

## Out Of Scope For Phase One

- Replacing the existing local/basic POS.
- Multi-branch management.
- Warehouse-to-warehouse transfer.
- Branch-to-branch transfer.
- Full accounting.
- Automated supplier purchasing.
- Waste tracking.
- Profitability calculations.
- Delivery-app reconciliation.
- Production-critical POS SQL Server integration.

## Deferred Capabilities

These may be considered in later phases:

- Waste tracking.
- Profitability and cost reporting.
- Delivery-app reconciliation.
- Read integration with POS sales/consumption data.
- More advanced reports.

## Rule

Any request listed here must not enter phase one unless the project scope is formally changed and the impact is reflected in Charter, BRD, SRS, and RTM.
